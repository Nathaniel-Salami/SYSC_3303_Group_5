# Iteration 0 - SYSC 3303 Real Time Concurrent Systems - Winter 2020

Purpose: The purpose of this software is to cover the intuition behind synchronizing Java threads through the implementation of an elevator system utilizing 3 threads, handling file input through a data structure.
